Margdarshak – Job Market Trend Analysis & Skill Demand Prediction System
🚀 Overview

Margdarshak is a data-driven career guidance system that analyzes job market trends and identifies the most in-demand skills. It helps students, fresh graduates, and job seekers make informed career decisions using real-time insights and interactive dashboards.

🎯 Problem Statement

With rapid advancements in technology and post-pandemic changes, the job market is constantly evolving. Many individuals struggle to:

Identify high-demand job roles
Understand required skills
Find opportunities across locations

Traditional career guidance lacks real-time, data-driven insights, creating a gap between industry needs and individual skillsets.

💡 Solution

Margdarshak bridges this gap by:

Collecting and analyzing job market data
Identifying trending skills and roles
Visualizing insights using interactive dashboards
Providing data-driven career guidance
✨ Features
📊 Job Trend Analysis (2020–2035)
🔍 Skill Demand Insights
🌍 Location-Based Job Analysis
📈 Interactive Power BI Dashboards
🧠 Career Recommendations


🛠️ Tech Stack
Python – Data Collection & Processing
MySQL – Database Management
Power BI – Data Visualization
